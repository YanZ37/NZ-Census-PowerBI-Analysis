## Purpose & Policy Relevance

This project is designed to demonstrate **applied demographic and workforce analysis** using official census data.  
It addresses questions relevant to:

- labour-market and workforce planning,
- migration and settlement analysis,
- skills and qualification distribution,
- population ageing and structural change.

The dashboard is structured for **briefing, exploration, and high-level insight**, rather than technical modelling.

---

## Key Questions Explored

- How does population growth differ between **NZ-born and overseas-born groups**?
- How do **age structures** vary across birthplace groups?
- What shifts are occurring in **highest qualification attainment** over time?
- How are different population groups distributed across **occupation categories**?
- Are there signs of **upskilling or occupational polarisation**?
